# 3D Vector Operations Program

This C++ program performs various operations on 3D vectors, including calculating the norm, sum, angle, dot product, projection, and cross product of two 3D vectors.

## Description

This program demonstrates the following operations on 3D vectors:

1. **Norm**: Calculates the length or norm of a vector.
2. **Vector Sum**: Adds two vectors.
3. **Angle**: Calculates the angle between two vectors.
4. **Dot Product**: Computes the dot product of two vectors.
5. **Projection**: Calculates the projection of one vector onto another.
6. **Cross Product**: Computes the cross product of two vectors.

## Features

- Input vectors dynamically.
- Calculate the length of vectors.
- Perform vector addition.
- Calculate the angle between vectors.
- Compute the dot product.
- Find the projection of vectors.
- Compute the cross product.

## Usage

1. Clone the repository or download the `main.cpp` file.
2. Compile the program using a C++ compiler.
3. Run the compiled executable.
4. Follow the prompts to input the components of two 3D vectors.
