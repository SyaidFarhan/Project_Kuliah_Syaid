Dijkstra Algorithm Implementation in Java

This repository contains a Java implementation of Dijkstra's algorithm for finding the shortest path in a weighted graph.

## Features

- Efficient Dijkstra's algorithm implementation.
- Support for multiple destination nodes.
- User-friendly command-line interface for input.
- Demonstrates array and graph traversal techniques.
- Well-documented code with inline comments.

## Usage

1. Compile the `DijkstraAlgorithm.java` file using a Java compiler.
2. Run the compiled Java class. Follow the prompts to input the starting point and destinations.
3. The program will display the shortest paths and total distance traveled.

Example usage:
```java
public static void main(String[] args) {
    DijkstraAlgorithm.djikstra();
}
